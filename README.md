# Agenda de Contatos
#### Sobre
* Projeto de estudos com a intenção de realizar um CRUD de usuários de uma forma simples e eficiente, trabalhando com o banco de dados relacional e trabalhando um pouco de estilo com CSS. 

### Estilos
* Bootstrap e Font-Awesome. <https://cdnjs.com/>

### Banco de Dados 
* Conexão PDO com Mysql 

### Linguagens usadas durante o Projeto:
* PHP - Backend
* CSS - Estilo
* SQL - Banco de Dados
